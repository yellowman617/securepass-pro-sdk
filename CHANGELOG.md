# Changelog

All notable changes to the SecurePass Pro SDK will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-07-25

### Added
- Initial release of SecurePass Pro SDK
- Core password generation functionality
- Bulk password generation (up to 1000 passwords)
- Team management features (add, remove, update roles)
- Usage statistics and monitoring
- Comprehensive error handling and validation
- Support for all SecurePass Pro plans (Basic, Pro, Enterprise, Annual)
- Secure API key validation
- Timeout handling and request management
- Cross-platform compatibility (Node.js and browser)

### Security
- CSPRNG-based password generation
- API key format validation
- Request timeout protection
- Input sanitization and validation
