Installation

npm install securepass-pro-sdk

Quick Start

import SecurePassSDK from 'securepass-pro-sdk';

// Initialize the SDK
const sdk = new SecurePassSDK('your-api-key-here');

// Generate a secure password
const password = await sdk.generatePassword({ length: 16 });
console.log(password.password); // "K8#mN9$pQ2@vX7!"

// Generate multiple passwords
const bulkPasswords = await sdk.generateBulkPasswords(5, { length: 12 });
console.log(bulkPasswords.passwords); // Array of 5 passwords

API Reference

Constructor

new SecurePassSDK(apiKey, options)

Initialize the SDK with your API key and optional configuration.

generatePassword(options)

const password = await sdk.generatePassword({ length: 16 });

Generate a single secure password with specified options.

generateBulkPasswords(count, options)

const passwords = await sdk.generateBulkPasswords(10, { length: 12 });

Generate multiple passwords at once (up to 1000).

getUsage()

const usage = await sdk.getUsage();

Get your current usage statistics and plan limits.

Examples

Basic Usage

const sdk = new SecurePassSDK('your-api-key');

// Simple password
const password = await sdk.generatePassword({ length: 16 });

// Check usage
const usage = await sdk.getUsage();
console.log(`Used ${usage.usedGenerations}/${usage.passwordLimit} passwords`);

Team Management

// Get team info
const teamInfo = await sdk.getTeamInfo('team_12345');

// Add team member
await sdk.addTeamMember('team_12345', 'user@company.com', 'member');

// Update role
await sdk.updateTeamMemberRole('team_12345', 'user@company.com', 'admin');
